# how to store/install/uninstall all the packages of a project 

## 1. to freeze all installed package in a file
pip freeze>requirements.txt

## 2. to install all the packages of requirements.txt file 
pip install -r requirements.txt

## 3. to uninstall all the packages of requirements.txt file 
pip uninstall -r requirements.txt
